### 개인 공부 프로젝트

## 1. 개발을 해보자 !
퇴사 후, 목표 없이 공부를 하다보니 의욕도 줄어들고 방향성 조차 없어지면서 의미를 퇴색해가고 있었던 찰라에 선배님 한분이 개인프로젝트를 진행 해보시라는 말씀과 함께 시작되었습니다.
지금까지는 개념적인 부분으로만 공부해오던 기술들을 개인프로젝트를 하면서 적용 하기로 마음먹었습니다. 많은 부분이 부족할지 모르겠지만 선배님과 같이 짝프로그래밍 형식으로 열심히 해야겠습니다.


## 2. 개발을 시작하기 앞서..
실무 경험은 그렇게 길진 않지만 여러가지를 많이 접해본 결과... 객체지향적인 개념을 실제로 적용해본적은 드물었던 것 같습니다. 실제 OOP에 대한 개념이 진짜 적용이 가능할지에 대한 의문이 많지만 프로젝트를 통해 기술적인 부분 뿐만 아니라 이러한 부분들을 해소해 나갈 예정입니다. 

## 3. 프로젝트
프로젝트는 MSA 구조의 쇼핑몰입니다. 비지니스 로직보다는 기술적인 부분, 객체지향 개념와 디자인 패턴을 이용한 구조를 중점으로 할 예정입니다. 또한 생각보다 긴시간이 걸릴 것 같습니다. 하지만 개념적인 부분을 실제 프로젝트에 적용 해본다는 것은 정말 어려운 일이기에 이를 계기로 좀더 여러가지 관점으로 성장할 계기가 될 것 같습니다. 

## 4. 사용할 기술들
- Spring Boot
- Spring Data JPA
- QueryDSL
- JAVA 11
- Redis
- Postgersql
- Swagger 
- Spring Test Junit5 / Mockito
- Spring Cloud Netflix Juul / Ribbon / Eureka / Hystrix
- OAUTH 2 + JWT
- IntelliJ IDEA Ultimate (1월부터 사용예정)

## 5. 프로젝트 실행 환경 및 방법(추후 업데이트 예정)
	- Lombok와 QueryDsl 설정이 되있어야합니다.
